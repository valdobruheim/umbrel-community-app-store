# Prometheus Umbrel App


This UmbrelOS app installs **Prometheus**, an open‑source monitoring and metrics collection system.


## Features
- Metrics scraping and storage
- Persistent Prometheus data volume
- Fully isolated Umbrel app service


## Ports
- Prometheus UI: **9090**
