# Node Exporter Umbrel App


This UmbrelOS app installs **Node Exporter**, exposing OS and hardware metrics that Prometheus can scrape.


## Features
- Low‑resource system metrics exporter
- Designed to integrate with Prometheus


## Ports
- Node Exporter metrics: **9100**


## Dependency
This app **requires Prometheus** to be installed.
