# NerdQAxe Metrics Exporter Umbrel App

This UmbrelOS app installs **Json Exportert**, to exporter the NerdQAxe metrics that Prometheus can scrape.
The APIs were tested with esp-miner-factory-NerdQAxe++-v1.0.35

This should also work with any BitAxe

## APIs
* api/system/asic
* api/system/info

## Ports
- Node Exporter metrics: **7979**

## Dependency
This app **requires Prometheus** to be installed.
